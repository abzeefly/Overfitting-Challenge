To run the project :
1. Download the train and test dataset from kaggle from :
https://www.kaggle.com/c/dont-overfit-ii/data

2. Unzip the file into the folder where you have Final_submission.py file

3. Ensure you have the following packages :
    numpy
    pandas
    seaborn
    matplotlib
    sklearn
    xgboost
    imblearn
    mixtend

    To install a package on your machine- pip install <package>
    To install a package on cse machine- pip3 install <package>

4. Once you are sure you have all the packages run the file by :
    Home Machine : python Final_submission.py
    CSE : python3 Final_submission.py

5. To test the submission on kaggle go the link:

    https://www.kaggle.com/c/dont-overfit-ii/data

    Click on "Late Submission"
    Upload the file into the submission box
    Click on "Make Submission"


More about the project in the Project Report file.
